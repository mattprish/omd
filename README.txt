Привет!
Классы и функции в домашке раскиданы по разным файлам.
cnt.py - CountVectorizer, на выходе выдающий count_matrix 
tf.py и idf.py - функции для обработки count_matrix
tranformer.py - класс транформера для обработки count_matrix с использованием функций
и наконец
tfidf.py - класс TfidfVectorizer для обработки корпуса текстов 

в файле test есть необходимый импорт и прописана функция для запуска итогового векторайзера!
